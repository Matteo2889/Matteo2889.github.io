Custer's Last Fight

Objectifs principaux

Dans le cadre de cette SAÉ, nous avions donc pour objectif de créer le jeu vidéo de notre choix à partir d'une bibliothèque donnée.
Contraintres du projets:

  • Utiliser le framework fourni par l'IUT, donnant accès à des méthodes implémentant un système de collision, de coordonnées sur un jeu, de son, d'affichage de sprites à l'écran, etc
  • Être un jeu de type "retro" en 2D, approuvé par un enseignant durant un TP
  • Comporter une fenêtre d'édition de paramètres ainsi qu'une page de scores
  • Être disponible en français et en anglais, selon la langue du système d'exploitation

Description

Dans le cadre de cette SAÉ, nous avons décidé de réaliser Custer's Last Fight : un jeu d'action-aventure qui est basé sur un gameplay similaire aux Legend of Zelda originaux ou The Binding of Isaac, se passant dans des donjons à plusieurs salles contenant différents ennemis.
Nous avons réalisé un installateur pour le jeu, mais le jeu est la propriété de l'IUT donc je ne peux pas le partager directement ici.
